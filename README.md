# Memorai Web

Nuxt 4 frontend — interface do Memorai.

## Stack

- Nuxt 4.4 + Vue 3 (Composition API)
- TypeScript
- Tailwind CSS 4.2
- Pinia

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

## Scripts

| Comando | Descrição |
|---|---|
| `npm run dev` | Dev server |
| `npm run build` | Build de produção |
| `npm run preview` | Preview do build |
